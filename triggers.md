# Trigger statistics

Published triggers, windows, and pre-arranged financing across the Centre for Humanitarian Data's Anticipatory Action portfolio. **🔴 flags frameworks that have activated.** Return periods and activation probabilities are the trigger statistics published in each framework document; activation years are the historical (backtested) years each trigger would have fired.

!!! note inline end "How to read"
    *Return period* = the average interval between activations. *Activation probability* = the chance the trigger fires in a given year. *Overall* = the chance **any** trigger in the framework fires.
