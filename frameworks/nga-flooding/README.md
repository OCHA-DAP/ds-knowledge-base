# Nigeria flooding (nga-flooding)

Dual flood AA: riverine (Benue River, CERF+NHF) + flash (BAY states, NHF-only). Repo: `ocha-dap/ds-aa-nga-flooding`. OCHA/CHD runs the daily monitoring.

**Current version: [2025-08-11](2025-08-11.md)** (status: endorsed, valid 2025–2026).
