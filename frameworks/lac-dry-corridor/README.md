# Central America Dry Corridor drought (lac-dry-corridor)

**Multi-country** seasonal-drought AA over El Salvador, Guatemala, Honduras — one umbrella framework + per-country annexes, shared design, per-country calibration. Repo: `ocha-dap/ds-aa-lac-dry-corridor` (R/`targets`). The repo also carries **Nicaragua**, which is *not* in the official framework.

**Current version: [2025-02](2025-02.md)** (status: endorsed, 2024–2025).
