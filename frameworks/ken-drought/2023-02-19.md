# Kenya Drought — 2023-02-19

> The canonical trigger is the IFRC/KRCS Early Action Protocol EAP2022KE02 (`framework_doc`). The OCHA CHD repo (`code_ref`) is exploratory analysis proposing IOD-adjusted replacement thresholds, not yet endorsed or operational.

## Summary

Kenya's drought anticipatory action framework targets 23 arid and semi-arid land (ASAL) counties for the October-November-December (OND) short-rains season. The endorsed framework (IFRC/KRCS EAP2022KE02, approved October 2022) triggers early actions when the Kenya Meteorological Department (KMD) OND seasonal forecast shows a Standardized Precipitation Index (SPI) below -0.98 with at least 33% probability of occurrence in at least 3 of the 23 counties. Counties are then ranked by drought risk (INFORM index), dropping any already in alarm or emergency phase, and early actions activate in the top 3 remaining counties. The OCHA CHD repo is a parallel, exploratory analysis (December 2025 – June 2026) developing IOD-adjusted probability thresholds as a potential enhancement to the EAP trigger. The framework activated for the first time in September 2025 for the OND 2025 season.

## Method

**EAP (endorsed):** KMD issues OND seasonal forecasts at three lead times (approximately July, August, September). For each issue month, the forecast gives spatially gridded probability of SPI falling below -0.98 (RP5 event) across Kenya. The IBF (Impact-Based Forecasting) dashboard ingests the forecast, overlays IPC phase projections and NDMA drought phase monitoring, and identifies counties meeting the trigger threshold. Counties already in alarm or emergency drought phase are excluded (too late for anticipatory action), and the top 3 risk-ranked counties receive early actions.

**OCHA CHD repo (development):** Uses KMD CPT-format hindcast files (July, August, September issue months, 1997–2025) from Azure blob storage, ERA5 monthly precipitation from the Azure PostgreSQL DB for return-period validation, and weekly IOD Dipole Mode Index (DMI) from Bureau of Meteorology. The analysis clips KMD forecast grid to the 9 arid counties, computes the 50th-percentile probability at the "all arid counties" level, and compares to IOD-state-dependent thresholds. IOD is classified as negative when DMI ≤ -0.4 for ≥ 8 consecutive weeks.

## Trigger logic

- **Keys off:** KMD OND seasonal forecast (CPT format), probability of below-normal rainfall; IOD state from BoM DMI weekly series.
- **Decision rule (plain language, endorsed EAP):** If the KMD OND forecast shows ≥33% probability of SPI < -0.98 in ≥3 of the 23 ASAL counties, and NDMA monitoring shows those counties are not already in alarm/emergency drought phase, activate early actions in the top 3 risk-ranked counties.
- **Decision rule (repo proposal, not endorsed):** At each issue month (July, August, September), compute the median probability of below-normal rainfall across the 9 arid counties from KMD CPT hindcasts. If the 50th-percentile probability exceeds the IOD-adjusted threshold, activate. For RP5 (1-in-5-year): threshold is 40% under non-negative IOD, 35% under negative IOD. For RP3 (1-in-3-year): 60% under non-negative IOD, 50% under negative IOD. IOD state is assessed from BoM DMI: negative if DMI ≤ -0.4 for ≥8 consecutive weeks.
- **Activation structure:** Single activation window (OND season), three issue-month checkpoints (July, August, September). Earlier issue = higher lead time = lower threshold (for endorsed RP3/RP5 per-month thresholds in old constants) or IOD-adjusted flat threshold (new constants).
- **Calibration:** EAP uses SPI -0.98 (~RP5, 20th-percentile event) at ≥33% probability from KMD tercile forecasts. Repo proposes probability-of-below-normal thresholds calibrated against ERA5 return periods and validated on KMD hindcast data 1997–2016.
- **Authoritative source:** IFRC/KRCS EAP2022KE02 (`framework_doc`). The OCHA CHD repo is analysis only.
- **Operated by:** The live trigger is run by KRCS/KMD/NDMA, not OCHA CHD. The IBF dashboard is operated by KRCS with IFRC support.

**Counting rule:** Three checkpoints (issue months) within a single OND activation window, each with its own threshold. These share a single activation outcome (the same CHF 367,238 release), so they are one logical window with three checkpoints — but the README and constants expose them as separate threshold sets. ## Sources & repo completeness

- **Trigger taken from:** `framework_doc` (IFRC/KRCS EAP2022KE02, authoritative). The repo proposes a development-stage enhancement.
- **Repo completeness:** `{analysis: partial, deployed_code: lost}`. The repo contains hindcast review and IOD analysis notebooks but no pipeline, no scheduler, no monitoring code, and no deployment. The trigger itself runs in KRCS/IBF infrastructure outside this repo.
- **Discrepancies:** See frontmatter `discrepancies` field for the full tagged list. Most critical: (1) the endorsed EAP uses SPI ≤ -0.98 with ≥33% probability as the core trigger, while the OCHA CHD repo proposes probability-of-below-normal thresholds (35–60%) modulated by IOD state — these are different indicator formulations; (2) the repo covers only 9 arid counties vs the EAP's 23 ASAL counties; (3) the repo hardcodes only years 2005, 2010, 2016 as negative IOD years in calibration, but BoM data shows 2020, 2021, 2022, 2025 also had persistent negative IOD events meeting the criterion, so the calibration dataset is potentially incomplete.

## Monitoring

No automated monitoring pipeline in the OCHA CHD repo. The live monitoring is performed by KRCS using the IBF (Impact-Based Forecasting) dashboard, which ingests KMD seasonal forecasts and NDMA drought monitoring data. NDMA publishes monthly drought bulletins. KMD issues OND seasonal forecasts from July onwards. Monitoring cadence for the EAP: continuous NDMA surveillance + seasonal checkpoint at each KMD forecast issuance (July, August, September).

## Historical activations

**Activation 1 (2025-09-12):** KRCS activated EAP2022KE02 for the OND 2025 season. The KMD forecast showed conditions driven by a negative Indian Ocean Dipole meeting the EAP trigger (SPI < -0.98 with ≥33% probability in ≥3 ASAL counties). This was the **first ever activation** of the EAP, roughly 3 years after approval (October 2022). The OCHA CHD exploratory analysis began in December 2025, one month after this activation.

## Key decisions & rationale

**Why IOD modulation?** The OND short rains in Kenya are strongly influenced by the Indian Ocean Dipole. Negative IOD (warm western Indian Ocean) suppresses rainfall over East Africa and significantly increases drought probability. The OCHA CHD analysis shows that IOD state explains substantial forecast variability across lead times (notebook 01.3): lead-time forecast variability is highest in negative IOD years, meaning forecasts issued in July/August are more reliable in those years. Lowering the trigger threshold in negative IOD years increases sensitivity (catching more true droughts) at the cost of some specificity. The analysis recommends: non-negative IOD threshold 40% (RP5) / 60% (RP3); negative IOD threshold 35% (RP5) / 50% (RP3).

**Why 9 counties vs 23?** The repo focuses on the 9 arid counties (Baringo, Mandera, Isiolo, Wajir, Garissa, Marsabit, Turkana, Tana River, Samburu) which are more consistently drought-prone and have better KMD forecast coverage. The EAP covers all 23 ASAL counties for operational flexibility, including agro-pastoral counties with different rainfall dynamics.

**Why KMD CPT hindcasts?** KMD issues seasonal forecasts in CPT (Climate Predictability Tool) format, providing gridded probabilistic forecasts. These are more spatially specific than tercile-only products and allow county-level analysis. The repo parser.py handles the CPT file format.

**Why ERA5 for validation?** ERA5 provides a 1997–2025 gridded observational baseline against which KMD forecast skill and return periods can be validated. SPI is computed from ERA5 OND precipitation sums at county level.

## Open questions / known issues

- The OCHA CHD IOD-adjusted thresholds are development-stage and not yet incorporated into the operational EAP or IBF dashboard. When (and whether) these thresholds will replace the current SPI-based EAP trigger is not documented.
- The repo hardcodes negative IOD years as 2005, 2010, 2016 for calibration but the BoM DMI data shows 2020, 2021, 2022, and 2025 also meet the persistence criterion (≥8 weeks at DMI ≤ -0.4). The calibration should be revisited to include these years.
- The IOD state determination for operational use is not codified: the DMI threshold (≤ -0.4 for ≥8 weeks) is clear in the analysis but there is no code specifying exactly when in the forecast cycle (before July, before August, before September forecast) the IOD state is assessed to select the threshold.
- The repo loads hindcast blobs from the `stage="dev"` slot but reads ERA5 from the `stage="prod"` database (not a uniform dev deployment). No production/scheduled pipeline exists regardless.
- The 2025 activation targeted Kwale, Kilifi, and Kitui counties — notably, these are not among the 9 "arid counties" the OCHA CHD repo focuses on. This reflects the EAP's broader 23-county scope and risk-ranking process, suggesting the OCHA CHD analysis may be under-scoping its geographic target.
- WFP's separate AA activation (Marsabit and Wajir) runs on a different trigger framework. The relationship between the KRCS EAP, WFP AAP, and the OCHA CHD analysis is not documented.
