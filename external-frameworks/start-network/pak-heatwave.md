# START — Pakistan heatwave

## Summary
Start Network's anticipatory action system for heatwaves in Pakistan, run through the
READY Pakistan hub (originally hosted by Welthungerhilfe Pakistan) since a 2019 pilot.
An operational model forecasts heat-index conditions for a recurring set of cities and
districts across Sindh, Punjab and Balochistan; crossing a site-specific threshold
releases pre-positioned funds — since 2022 sourced through Start Ready, the network's
pooled catastrophe-risk-financing facility — to local implementing NGOs for early
actions like cooling stations, water points, cash, and health-facility support.

## Trigger
An operational heatwave model built on NOAA's Global Forecast System (GFS) monitors
heat-index conditions at recurring sites (Karachi, Larkana, Multan, Sibi, Jacobabad,
Nawabshah, and others added over time). Each site has its own calibrated threshold; the
best-documented is Karachi's: a forecast maximum temperature above 42°C for 2
consecutive days with the daily minimum above 30°C, matching the Karachi city
commissioner's own emergency heat threshold. In the 2026 Karachi/Port Qasim activation,
the model gave 6 days' lead time (20 May forecast for a 26-27 May event, 45.3°C
max/31.4°C min). The original 2019-21 pilot used a simpler design — temperature ≥3°C
above the seasonal average for 3 consecutive days, forecastable up to 10 days out —
and it is not documented in public sources exactly how/when each site moved to the
current GFS heat-index model.

## Funding & scope
Piloted from 2019 by Welthungerhilfe Pakistan under READY Pakistan, funded directly by
Start Network. The first heatwave activation (2022, Sibi district, Balochistan) drew on
a ~£48,000 ($62,777) Start Network grant backed by Germany's Federal Foreign Office.
From 2022, the system runs on **Start Ready**, Start Network's pooled catastrophe-risk
facility (launched at COP26 2021; first risk pool live May 2022 with £3m across 6
countries). It is now on **Risk Pool 3**, covering 8 countries (Philippines,
Bangladesh, Madagascar, Senegal, Somalia, Zimbabwe, DRC, Pakistan) against 13 climate
hazards for roughly 688,000 people, stretching £5.9m of capital to £9.7m through
pooling — public reporting does not break this down by country or hazard. Recurring
target locations include Karachi City, Larkana, Multan, Sibi, Jacobabad, Nawabshah,
Dadu, D.I. Khan, Layyah and Jaffarabad. The Anticipation Hub's global-map inventory
separately records $689,655 pre-arranged / 240,000 people targeted for this framework
record, which does not cleanly reconcile with other Hub-reported figures (see
`extra.schema_strain`).

## Activations
- **2022, Sibi (Balochistan)** — first-ever heatwave activation of the system;
  ~£48,000 ($62,777) Start Network grant, backed by Germany's Federal Foreign Office,
  delivered via Welthungerhilfe Pakistan.
- **May 2026, Karachi/Port Qasim** — a 20 May forecast of a 26-27 May 2-day heatwave
  (~45.3°C max/31.4°C min) crossed the Karachi threshold; released £36,000 to Action
  Against Hunger (ACF) and HANDS, part of a £100,000 pre-allocation split across 4
  Karachi-area regions.
- **May-June 2026, Nawabshah/Dadu/Jacobabad** — Start Ready Risk Pool 3 activation
  (act-00000043) triggered by forecasts of prolonged extreme heat; early action also
  delivered in Sibi, Multan, D.I. Khan, Layyah, Jaffarabad and Larkana by BSDSB, Mercy
  Corps, PRDS, Doaba Foundation, Azat Foundation and LHDP.
- **Near-miss, not an activation of this system:** on 2017-04-18, Start Members
  submitted a Start Fund heatwave alert (037) ahead of Pakistan's deadly April 2017
  heatwave (temperatures to 50°C); the Allocation Committee declined to fund it despite
  acknowledged need. This predates the anticipatory pilot (2019) and the current system.
