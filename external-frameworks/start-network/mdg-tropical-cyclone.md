# START — Madagascar tropical cyclone

## Summary
Start Network's anticipatory action (AA) mechanism for tropical cyclones in Madagascar,
funded through the pooled **Start Ready** risk-financing facility (with **Start Fund**
rapid alerts as a secondary channel) and implemented by a rotating set of Start Network
member agencies (CARE International, Catholic Relief Services, Humanity & Inclusion,
Medair, Save the Children, Action Against Hunger, Doctors of the World, Welthungerhilfe
across 2022–2024). On a Météo-Madagascar forecast of sustained winds ≥166 km/h within
72 hours of landfall, pre-positioned funding releases to members to run pre-agreed
early-action plans. Active every cyclone season since 2022, with confirmed activations
for Batsirai and Emnati (2022), Freddy (2023), Gamane (2024), and Fytia (Jan 2026).
Runs independently of, but in the same seasons and on a similar meteorological
threshold as, the separate OCHA/CERF collective Madagascar-storms framework.

## Trigger
A Météo-Madagascar forecast of a tropical cyclone with sustained winds of ≥166 km/h,
issued up to 72 hours before predicted landfall, triggers Start Network's cyclone
model — currently part of Start Ready Risk Pool 3. Start Network describes the
trigger-to-disbursement process as compressed to roughly 5 hours so member agencies can
be operational as fast as possible once the threshold is met. (This is distinct from,
but numerically identical to, the ≥166 km/h/100 km-of-coast "Scenario 3" threshold used
by the OCHA/CERF collective framework and WFP's own Madagascar AA line — see
`coordination` in the frontmatter.)

## Funding & scope
The Anticipation Hub inventory (fetched 2026-07-10) lists ~US$1.44M pre-arranged and
~35,324 people targeted for this framework, though these figures are close to — and may
simply reflect — the most recent (2024) activation rather than a fixed standing
envelope (see `extra.schema_strain`). Funding runs through two named instruments:
**Start Ready** (the multi-country pooled anticipatory financing mechanism Start
Network has run since 2022, currently in its third risk pool) and the **Start Fund**
(Start Network's faster rapid-allocation mechanism, used for both anticipatory and
reactive alerts). Actual per-cyclone disbursements recorded publicly range from
~£216,000 (Emnati, 2022) to ~£1.13M (Gamane, 2024) — see Activations.

## Activations
- **2 Feb 2022 — Cyclone Batsirai**: £419,046 (~US$562,000), ~41,559 people reached.
- **22 Feb 2022 — Cyclone Emnati**: £216,155 (~US$290,000), ~220,188 people reached.
- **21 Feb 2023 — Cyclone Freddy** (first Madagascar landfall, near Mananjary):
  £700,000 (~US$844,000) plus £170,000 from National Reserves for preparedness,
  ~42,409 people reached. Part of Start Ready's second risk pool (7 countries,
  May 2023–April 2024), which also covered a Feb 2024 Madagascar drought activation.
- **26 Mar 2024 — Cyclone Gamane**: £1,130,000 (~US$1.43M) released; the Anticipation
  Hub write-up describes pre-landfall implementation as limited, reading as a partial
  rather than full activation.
- **29 Jan 2026 — Cyclone Fytia**: Start Ready activated ahead of forecast landfall
  affecting Marovoay district, Boeny region; ~£430,000 disbursed for food, water,
  health care, emergency shelter and household items in Soalala and Marovoay ahead of
  landfall (31 Jan 2026). Sourced from Start Network's own social-media
  announcement only — no formal report was found.
- Public reporting on the concurrent Feb 2026 Cyclone Gezani is not cleanly
  disaggregated between Start Ready and Start Fund, or between Fytia and Gezani
  spending (figures cited include EUR 718,000 combined Start Ready funding and a
  separate US$1.46M Start Fund figure for Gezani) — not recorded as a distinct
  activation pending a clearer primary source. The separate OCHA/CERF collective
  framework (`frameworks/mdg-storms`) released its own full US$3M CERF envelope for
  the same storm.
